# Daddy: Deep-leArning for Depression DiscoverY 

### 1. Quick Start

```shell script
# clone the project 
git clone git@github.com:celsofranssa/Daddy.git

# change directory to project folder
cd Daddy/

# Create a new virtual environment by choosing a Python interpreter 
# and making a ./venv directory to hold it:
virtualenv -p python3 Daddy/

# activate the virtual environment using a shell-specific command:
source ./Daddy/bin/activate

# install dependecies
pip install -r requirements.txt

# setting python path
export PYTHONPATH=$PATHONPATH:<path-to-project-dir>/Daddy/

# (if you need) to exit virtualenv later:
deactivate
```
